## 这是一个管线占位章节

这一页**不是**正式教学内容，它的唯一职责是验证 MDX 内容管线端到端跑通：
从仓库里的 `.mdx` 文件，经 frontmatter 解析与动态路由，最终被 SSG 渲染成一张静态页。

正式的章节正文、交互 Demo、代码对照与出处声明，会在后续里程碑按
`docs/chapter-spec.md` 的规范逐节补齐。本页只用最基础的 Markdown 语法。

### 管线验证清单

- frontmatter 被类型安全解析（缺字段构建期报错）
- `getAllChapters` / `getChapter` 提供内容索引
- `generateStaticParams` 为每章产出一张静态页
- 标题、段落、列表、代码块都能正常渲染

下面几段代码块用来验证 shiki 高亮（github-dark）对多语言可用。GLSL 顶点着色器：

```glsl
#version 330 core
layout (location = 0) in vec3 aPos;
void main() {
  gl_Position = vec4(aPos, 1.0);
}
```

C++ 侧创建 VAO：

```cpp
unsigned int VAO;
glGenVertexArrays(1, &VAO);
glBindVertexArray(VAO);
```

WebGL2-TS 侧拿到上下文：

```ts
const gl = canvas.getContext("webgl2");
if (!gl) throw new Error("WebGL2 not supported");
```

数学方面，裁剪空间到屏幕的透视除法是逐分量除以 $w$ 分量；投影矩阵把视锥映射到
NDC 立方体，可写成下面这个独立公式块：

$$
P = \begin{bmatrix}
\frac{1}{a\tan(\theta/2)} & 0 & 0 & 0 \\
0 & \frac{1}{\tan(\theta/2)} & 0 & 0 \\
0 & 0 & -\frac{f+n}{f-n} & -\frac{2fn}{f-n} \\
0 & 0 & -1 & 0
\end{bmatrix}
$$

到这里，如果你能在浏览器里看到这页标题、正文、彩色代码块与公式，说明管线已经通了。
